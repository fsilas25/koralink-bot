KoraLink AI Chatbot
===================

Project Name:
-------------
KoraLink AI Assistant

Description:
------------
This chatbot helps users learn about KoraLink, the KoraLink Marketplace,
Digital Community Champions, products, training, commissions, and digital services.

Technology Stack:
-----------------
- Python
- Flask
- HTML
- CSS
- JavaScript

Features:
---------
✓ Website chatbot interface
✓ KoraLink FAQ system
✓ Agent information
✓ Marketplace information
✓ Commission information
✓ Product information
✓ Training information
✓ Inventory information
✓ Sales tracking information

Installation:
-------------

1. Create virtual environment:

   python -m venv venv

2. Activate environment:

   Windows:
   venv\Scripts\activate

3. Install requirements:

   pip install -r requirements.txt

4. Download spaCy model:

   python -m spacy download en_core_web_sm

5. Train the chatbot on flask Run the Flask server:

   python app.py

6. Open browser:

   http://127.0.0.1:5000
